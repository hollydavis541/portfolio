Pocket Gallery

Stack: Node, Express, SQL, EJS, HTML, CSS
Pocket Gallery is a full-stack web application that utilizes the Harvard Art Museum API. Users can search for works, save their favorites to custom galleries, filter works by gallery, and update or delete works. In addition, when a user views the details for a specific work, the app uses Imagga's color-detection API to change the matte background of the painting (and the button colors on the details page) to match the dominant colors in the painting. 

Cosmic Arcade

Stack: JavaScript, HTML, CSS
Cosmic Arcade is a mobile-first, 80s-themed site that features a fun rock-paper-scissors-lizard-Spock game for users of all ages. The landing overlay page features a Tron-like grid and player name input. The game play page features neon SVG graphics (icon glow created in Photoshop). The user chooses their "weapon" which triggers the computer (Sheldon) to display its randomized choice. The scoreboard increments based on the game logic and displays custom messages to the user based on the outcome of each round.

BusMall Research

Stack: JavaScript, HTML, CSS, Chart.js
A market research page to help the fictional BusMall startup (like SkyMall) determine what novelty gift items to ultimately publish in their catalog. The user chooses the product they would be most likely to purchase. After finishing a round of choices, voting statistics display on the left sidebar and in a dynamically-rendered chart below.

about me

I am a curiosity-driven software developer who is eager to put my professional and technical skills to use for a company that is solving interesting problems. I have a degree in graphic design, but ended up working as a legal assistant for several years. Wanting to get into tech, a few years ago I found a position as a tech support agent for a WebRTC-based agile software startup. While working there, I gained exposure to the development cycle and I was intrigued because I could see that it was about the the interplay between problem solving and creativity.

I am excited to now be learning new programming languages and tools myself. It is inherently interesting to me to be able to understand what’s going on “under the hood” of the software we use every day, and to then use this knowledge to create useful applications. I am deeply driven by curiosity and compassion and I would love to employ these qualities in service of your company’s mission.
